# 🌾 Pocket Farm Log

A lightweight, mobile-friendly web application designed to track farm activities, health checks, and harvests in real-time. Built with a clean, modern interface using Tailwind CSS.

## 🚀 Features
* **Activity Logging:** Quickly log health/veterinary checks or harvest data.
* **Quick Notes:** Add specific details for each entry (e.g., "North pasture sheep fed").
* **Recent Activity Feed:** View your latest logs at a glance.
* **Clean UI:** Minimalist design with an emerald-and-stone color palette for easy outdoor reading.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **Tailwind CSS:** Utility-first styling for a responsive layout.
* **JavaScript:** Logic for handling form submissions and data display.

## 📂 Project Structure
* `index.html` - The main dashboard and form.
* `style.css` - Custom styling and Tailwind imports.
* `script.js` - Interactive functionality and log management.
